# 📝 Todo List App

## 시스템 소개
React로 만든 간단한 할 일 관리 앱입니다.

## 기능 소개
- 할 일 추가
- 할 일 완료 체크
- 할 일 삭제

## 기술 스택
- React
- AWS S3
- GitHub Actions
- AWS Amplify

## GitHub Actions CI/CD 환경
- main 브랜치에 push하면 자동으로 AWS S3에 배포됩니다.
- GitHub Secrets에 AWS 키를 등록하여 보안을 유지합니다.
